# 🎯 Java Number Guessing Game

A simple Java console game where the player tries to guess a randomly generated number between **1** and **100**. This project was built to practice Java fundamentals, including user input, loops, conditional statements, random number generation, and game logic.

## Features

* Randomly generates a number between **1** and **100**
* Allows the player to make unlimited guesses
* Provides feedback if the guess is too high or too low
* Displays the correct number when guessed
* Tracks and displays the total number of attempts
* Console-based gameplay

## Technologies Used

* Java
* IntelliJ IDEA

## How It Works

1. The program generates a random number between **1** and **100**.
2. The player enters a guess.
3. The program checks whether the guess is too high, too low, or correct.
4. Feedback is displayed after each guess.
5. The game continues until the correct number is guessed.
6. Once the correct number is found, the program displays the number of attempts taken.

## Example

```text
Number Guessing Game
Guess a number between 1-100

Enter a guess: 75
TOO HIGH! TRY AGAIN

Enter a guess: 30
TOO LOW! TRY AGAIN

Enter a guess: 48
CORRECT! The number was 48
Number of attempts: 3
```
